## ğŸš€ Getting Started

These projects are simple HTML/CSS/JavaScript applications that run directly in the browser.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code, Sublime Text, etc.) if you want to explore the code

### Running the Projects

1. Clone this repository:
```bash
git clone https://github.com/FAZ110/MyTraining.git
```

2. Navigate to the project folder:
```bash
cd MyTraining
```

3. Open any project's `index.html` file directly in your browser, or use a local development server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

4. Browse to `http://localhost:8000` and navigate to individual project folders
